# lista-telefonica-angularjs
App de lista telefonica simples com AngularJs, utilizando as principais diretivas:


- ng-repeat, ng-model
- ng-disabled
- ng-options
- ng-class e ng-style
- ng-show e ng-hide
- ng-include
- $valid e $invalid
- $pritine e $dirty
- $minlength e $maxlength
- ng-pattern
- ng-messages
- filters
